# CNProjectSem4
UDP System using CRC or checksum to detect transmission errors

# How to use:
- Open terminal
- Run the python file ui.py
- Start the server from the ui
- Browse the txt file you wish to use for transmission (or for reference you can use the userInput.txt from the repository)
- You can analyse the summary of client side on the ui
- You can also analyse the output in the output.txt file of the repo (or if you wish to send the output to some other file you are free to change the path in codebase)
